This above files consists of:
myenv: this zip file is the activation file which consists of all the libraries required for the Django web application.
WebApplicationCode: All the data and codes related to the web application including the Machine learning model and dataset required to run it also the txt file generated from hardware.
Hardware Configuration Codes: This consists of files used to configure the hardware.
Before compling the web application we have to activate the myenv zip file available at https://drive.google.com/file/d/11TzjYE07tUlGk1TSEKRgHTTmYTkF9T1b/view?usp=drive_link
This can be done using command "conda activate myenv"
